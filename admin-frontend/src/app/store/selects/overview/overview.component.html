🚀 Introducing Workspace Select – Seamless Workspace Selection for Your Angular Apps!

Building a SaaS or enterprise application with multiple workspaces? Workspace Select is a ready-to-use Angular Material component that provides an intuitive and user-friendly way to select workspaces.

  • ✅ Fully integrated with Angular Material – modern design aligned with Material 3 and Tailwind
  • ✅ Highly customizable – easily adapts to any workspace selection logic
  • ✅ Supports async data – load workspaces dynamically
  • ✅ Perfect for SaaS applications – simplifies workspace switching for users
